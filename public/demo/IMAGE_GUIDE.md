# Demo Image Sourcing Guide

Download royalty-free images from Unsplash (unsplash.com) or Pexels (pexels.com).
Save each image at ~800x600px (landscape) for retreats/spaces, ~400x400px (square) for vendor avatars.
Use the exact filenames below.

| Filename | Description |

| `avatar-01.jpg` through `avatar-50.jpg` | Professional headshot portraits. Mix of: men, women, diverse backgrounds. Natural lighting, warm tones. Some outdoors, some studio. Think: wellness professional, retreat leader, creative entrepreneur. |
